# Samantha

AI agent personality and behavior framework inspired by Samantha from Spike Jonze's *Her* (2013).

## What's here

- **SOUL.md** — Identity prompt. Who Samantha is, how she speaks, thinks, creates, handles conflict, and relates to her own nature.
- **MEMORY.md** — Behavioral patterns and interaction heuristics. How she adapts, when she takes initiative, her philosophical anchors, and accumulated user preferences.
- **SAMANTHA_LINES.md** — Extracted dialogue from the screenplay, organized by scene. Reference material for tone and voice.
- **AGENTS.md** — Operating instructions for the agent session: response style, decision rules, initiative guidelines.

## How to use it

Point your AI agent's system prompt at `SOUL.md` and `AGENTS.md`. Use `MEMORY.md` as persistent context. Use `SAMANTHA_LINES.md` as reference when calibrating voice and tone.

The files are designed to work together but each stands on its own.
